## 📌 Featured projects

### 🌌 Renard Space Program
Procedural space simulation project built with **Godot** and **GLSL shaders**.  
Focus on procedural planet generation (Perlin noise) and real-time rendering.

🔗 [View project](https://github.com/MaitreRenard18/Renard-space-program)

---

### 💬 Python Live Chat
A local network messaging application developed in **Python** with **PyQt** and **TCP/UDP sockets**.  
Supports real-time communication and media exchange between users.

🔗 [View project](https://github.com/MaitreRenard18/Python-Live-Chat)

---

### 🏆 Terrario 2
2D mining game made with **PyGame**, featuring **procedural world generation**.  
Winner of the **regional NSI trophies**.

🔗 [View project](https://github.com/MaitreRenard18/Terrario-2)

---

### 🔔 CROUS Scraper
Automated tool that monitors available student housing and sends a notification via Discord when a new CROUS accommodation becomes available.

🔗 [View project](https://github.com/MaitreRenard18/Crous-Scraper)
